Bing Hintergrundbild-Crawler v1.0 by kill0rz

Stand: 26.05.2016

+ Installation

Wir laden alle Dateien aus dem ZIP-Archiv hoch und geben den beiden Ordnern Schreibrechte.

+ Anwendung

Zum Aufrufen der Bilder wird täglich die crawler.php (z.B. per Cronjob) aufgerufen. Die index.php gibt uns dann eine Gallerie der sich auf dem Server befindlichen Dateien zurück.

+ Lizenz

Der Crawler wurde unter kill0rz Unilicence v1.0 veröffentlicht. Diese liegt den Archiv bei.
